# 不可结缘 (Yokai Game)

这是一款以「奇幻日常感 × 万物有灵」为核心的像素风格 2D 收集 RPG 网页小游戏。

## 🎮 游戏特色
- **单元剧叙事**：在不同的场景下遭遇妖怪，体验他们背后淡淡忧伤又治朗的故事。
- **动态小游戏**：基于收集进度的动态难度递增机制，每一只妖怪都有专属的小游戏（目前包含“雨女”关卡：接大眼泪与躲避雷珠）。
- **生成式游戏音乐**：完全不依赖外部音频文件，使用代码 (Web Audio API) 实时生成像素风格的背景音乐及音效。
- **收集图鉴**：收集妖怪的名字、立绘、长故事与信物。

## 🚀 如何运行

本项目是一个纯前端（HTML + CSS + JavaScript）项目，**不需要安装任何依赖或构建工具**。

### 方法一：直接双击打开（最简单）
1. 找到项目根目录下的 `index.html` 文件。
2. 双击该文件，它会自动在你的默认浏览器（如 Chrome, Edge, Safari）中打开。
3. 即可直接开始游玩！

### 方法二：使用本地服务器（推荐）
如果你遇到了一些浏览器安全限制（例如 localStorage 或某些高级特性无法正常运作），可以使用简单的本地服务器来运行：

**如果你有 Python 环境：**
打开终端（Terminal），进入游戏目录并运行：
```bash
python3 -m http.server 8000
```
然后在浏览器中访问：`http://localhost:8000`

**如果你使用 VS Code：**
1. 在 VS Code 中打开本项目文件夹。
2. 安装 **Live Server** 插件。
3. 右键点击 `index.html`，选择 **"Open with Live Server"**。

## ⌨️ 操作说明
- **剧情推进**：可以通过 `鼠标点击`，或者按键盘的 `空格键 (Space)` / `回车键 (Enter)` 推进文本。
- **人物移动**：非对话状态下，按键盘的 `D`（向右走）或 `A`（向左走）。
- **UI 交互**：右上角有两个悬浮按钮，分别控制 **背包(收集册)** 的展开/收起，以及 **BGM 音效** 的开启/关闭。
- **雨女关卡**：移动鼠标控制雨伞的位置，接住**蓝色的「大眼泪」**，接住**白色的「小眼泪」**可以获得附加分，**躲避黄色的「雷珠」**。

## 📁 目录结构
- `index.html` - 游戏入口
- `css/` - 界面样式与自定义像素UI
- `js/` 
  - `main.js` - 核心运行逻辑与场景切换
  - `scenes/` - 各个场景（主菜单、探索、剧情、小游戏、结算）的代码
  - `data/` - 妖怪的故事内容与数值配置
  - `utils/` - 自定义绘图引擎与 Web Audio 音乐生成器
  - `collection.js` - 图鉴系统逻辑
